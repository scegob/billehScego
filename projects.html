Projects

Environmental Science

Working with my Professor, I oversaw the development of an Environmental Science webpage customized for our course syllabus. This extensive platform serves as an interactive study guide for both present and future students, and it was painstakingly designed to encompass the range of topics we covered throughout our sessions. I have methodically prepared a reservoir of resources, including educational articles, visual aids, and supplemental materials, to facilitate learning and memory. This has been achieved via diligent research and effort. It gives me great satisfaction to know that my lecturer still uses this website as a starting point when he or she is trying to explain environmental studies to new students.

Tic Tac Toe

I took on the challenge of creating a traditional Tic Tac Toe game as part of my quest to improve my JavaScript abilities, and the result is a project that I am glad to have in my portfolio. I was able to go deeply into the language with this project and use it to create a dynamic and engaging gaming experience. I concentrated on decomposing the game logic into digestible chunks and creating the framework and functionality from the ground up. I was able to hone my problem-solving skills and learn more about the complexities of JavaScript, including how it can handle game dynamics and user interactions. Creating this project demonstrated my technical proficiency as well as my commitment to lifelong learning and real-world programming application.
